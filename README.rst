Requirements
============
- Python >= 3.4
- django >= 2.1.3


Usage
=====

Start the server and play! Game files would be download on-demand.
You can download them all first to save time.

To start the server
-------------------
``python3 setup.py start_server``

To download all games
---------------------
``python3 setup.py download_all``
